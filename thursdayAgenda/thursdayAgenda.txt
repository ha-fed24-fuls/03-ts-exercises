💻 Code-Along: DOM manipulation

🛠 Förbered repo (en gång, om inte redan klart)

I projektroten:
1) Skapa package.json + scripts med Vite/TS:
npm init -y

2). Lägg till Vite + TypeScript som dev-deps:
npm i -D vite typescript

3). Sätt scripts och metadata utan att öppna filen:
npm pkg set name="fed24gb-fullstack-21-aug" private=true type=module
npm pkg set scripts.dev="vite" scripts.build="vite build" scripts.preview="vite preview"

4) Skapa/uppdatera tsconfig (om den saknas):
npx tsc --init

tsconfig.json – se till att du har DOM-typer:
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "Bundler",
    "strict": true,
    "lib": ["ES2020", "DOM"]
  }
}
